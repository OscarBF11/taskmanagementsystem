Sistema sencillo para la gestión de tareas.

Descripción

Este software ha sido desarrollado en menos de 72 horas con el propósito de proporcionar una solución simple y eficaz para un escenario de uso práctico: la administración de tareas entre múltiples usuarios de manera independiente.

Consideraciones
  • Los administradores pueden gestionar toda la aplicación y el resto de usuarios(Necesitan tener todos los roles incluido administrador).
  • Es necesaria conexión a internet para poder cargar los estilos al compilar.
  • Todo el diseño excepto las explicaciones han sido escritas en inglés.
Funcionalidades
  • Crear, editar y eliminar tareas.
  • Asignar tareas a usuarios específicos.
  • Gestionar roles de usuario.
  • Visualizar todas las tareas en un solo lugar.
